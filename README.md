# 🎵 Music Playlist Manager

This project is a Java-based application that simulates a music playlist using a Doubly Linked List data structure. This application will allow users to efficiently manage their music playlist in a similar way to how they would use a real music player.

## 🚀 Features
- Add Songs to Playlist
- Delete Songs from Playlist
- Search Songs in Playlist
- Display Playlist
- Play Current Song
- Navigate to Next Song
- Navigate to Previous Song
- Shuffle Playlist Randomly

## 🧠 Concepts Used
- Data Structure: Doubly Linked List
- Object-Oriented Programming (OOP)
- ArrayList and Collections: Shuffle Functionality

## ⚙️ How It Works
- Each Song is stored as a Node in a Doubly Linked List data structure.
- Playlist data structure has a Head pointer, a Tail pointer, and a Current pointer.
- Navigation is achieved using Next and Previous pointers.

## 🛠️ Technologies Used
- Java

## ▶️ How to Run
- Compile the program using:
  javac MusicPlaylistManager.java
- Run the program using:
  java MusicPlaylistManager
